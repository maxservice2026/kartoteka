# Kartoteka

Webova aplikace pro kartoteku kosmetickeho salonu.

## Spusteni

1. `npm install`
2. `npm start`
3. Otevri `http://localhost:8788`
4. Pri prvnim spusteni zaloz administratora, pak se prihlas

## Data

SQLite databaze je v `data/kartoteka.sqlite`.
Zaloha a obnova funguji pres JSON soubor ve webovem rozhrani.
Pri obnoveni se zneplatni prihlaseni a je potreba se znovu prihlasit.
